# Uruchamianie

1. Sklonować repozytorium.
2. Odpowiedni dataset wstawić do jednego z katalogów w ./datasets/ (same obrazki, bez podkatalogów)
3. Przejść do ./scripts
4. ```python launcher.py```
5. Reszta zgodnie z komunikatami z programu
